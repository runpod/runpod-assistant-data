# Why didn't I receive my referral bonus?

If you signed up using a referral link, clicked **Claim**, and completed the required deposit, but did not receive the referral bonus, there are a few common reasons this can happen.

## How referral bonuses work

To qualify for a referral bonus, you must:

- Sign up using a valid referral link
- Complete registration through that same referral flow
- Claim the referral during sign-up, if prompted
- Meet any required eligibility or funding conditions

If any part of this process is interrupted or changed, the referral bonus may not be applied.

## Common reasons a referral bonus is not applied

### A different referral link was already associated with the sign-up

This is one of the most common causes.

If you clicked a different referral link earlier, that referral code may still have been active in your browser session. Even if you later use another referral link, the original code may remain attached during registration.

As a result, the referral code you expected may not be the one that was actually applied to the account.

### The referral flow was not completed in the same session

The referral may not apply correctly if the sign-up process was interrupted or completed outside the original referral session.

This can happen when:

- the referral link is opened, but registration is completed later
- the browser session is interrupted
- cookies are cleared or blocked by your browser extension or anti-virus software
- sign-up is completed in a different browser or device
- the account is created outside the original referral flow

To receive the referral bonus, the referral link must remain correctly associated with the account throughout registration.

### The account was not eligible under referral program rules

Referral bonuses are intended for inviting new users to try Runpod.

An account may be considered ineligible if the system detects activity that does not align with the intended use of the referral program. This can include signals suggesting that both accounts may belong to the same user or that the referral was not a genuine invitation to a new user.

In such cases, the referral bonus will not be applied.

## Can a referral code be changed after sign-up?

No. Once an account has been created, the referral code associated with that registration cannot be changed manually.

## How to avoid referral issues

To improve the chances of receiving a referral bonus successfully:

- use the correct referral link from the start
- avoid opening multiple referral links before signing up
- complete registration in the same browser session
- avoid browser settings or extensions that block cookies
- make sure the referral is being used for a genuine new user

## Summary

If a referral bonus does not appear, it is usually because the wrong referral code was applied during sign-up, the referral session was interrupted, or the account did not meet eligibility requirements under the referral program.
